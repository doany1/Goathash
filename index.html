About Goat Hash

How it works?
Goat Hash uses a "Serverless Lookup" architecture. For MD5 and NTLM, we use High-Speed Sharding. The 14.3M database is split into 16 compressed chunks. When you search, the site only downloads the single tiny piece (~30KB) that contains your hash, making it instant even on slow connections.

Offline Scanning
For SHA1 and SHA256, we use a custom WebWorker engine. It streams a highly-compressed wordlist of 14.3 million entries into your browser's background thread, searching for a match on-the-fly. This allows massive searches without requiring multi-gigabyte server databases.

Privacy First
Because all processing happens in your browser, no one but you knows what you are searching. Our database is hosted for free on GitHub Pages using static file ranges, ensuring it stays online forever without costs.
